# yaziの使い方

基本の動き方

  ┌───────┬───────────────────────────────────────┐
  │ キー  │                 動作                  │
  ├───────┼───────────────────────────────────────┤
  │ h / l │ 親ディレクトリ / 子ディレクトリに移動 │
  ├───────┼───────────────────────────────────────┤
  │ j / k │ 上下移動                              │
  ├───────┼───────────────────────────────────────┤
  │ Enter │ ファイルを開く                        │
  ├───────┼───────────────────────────────────────┤
  │ q     │ 終了                                  │
  └───────┴───────────────────────────────────────┘

  vim のキーバインドそのままなので、vim を触ったことがあれば馴染みやすいです。

  よく使う操作

  ┌───────┬──────────────────────────────┐
  │ キー  │             動作             │
  ├───────┼──────────────────────────────┤
  │ Space │ ファイルを選択（複数選択可） │
  ├───────┼──────────────────────────────┤
  │ y     │ コピー                       │
  ├───────┼──────────────────────────────┤
  │ d     │ カット                       │
  ├───────┼──────────────────────────────┤
  │ p     │ ペースト                     │
  ├───────┼──────────────────────────────┤
  │ a     │ ファイル/ディレクトリ作成    │
  ├───────┼──────────────────────────────┤
  │ r     │ リネーム                     │
  ├───────┼──────────────────────────────┤
  │ D     │ 削除                         │
  ├───────┼──────────────────────────────┤
  │ /     │ 検索                         │
  └───────┴──────────────────────────────┘
